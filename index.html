Course Recommender Program

This is a project that I made for my INST326 class, where I developed a course recommender program using Python that searches for courses based on keywords and department names. Led a team of two members in implementing the program and utilized SQLite as the database management system to store course information and sections. Designed and implemented web scraping functionality using Selenium to extract course data from a website.

Motor Vehicle Collisions Database

In collaboration with a team, I'm designing a relational database to capture NYPD-reported motor vehicle collisions since July 1st, 2012. We're analyzing a dataset of over 100,000 rows, using SQL to focus on key collision factors and vehicle types. Our final deliverable will include a report outlining the database's development, challenges, strengths, and suggestions for future enhancements.
