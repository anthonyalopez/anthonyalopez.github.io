Course Projects

Course Recommender Program

A Python-based course recommendation system that filters courses by department and keywords. The system was built using SQLite for database management and incorporated web scraping via Selenium to gather course information directly from the university’s site. Led a small team to implement and test the system, ensuring accurate and up-to-date course recommendations.

End-to-End IoT System Model

A network model simulation using Cisco Packet Tracer, designed to configure IPv4 addressing for home and mobile LANs, ISP tiers, and a cloud data center. The project included network testing for connectivity through MAC address population, routing tables, and successful pings. Integrated IoT devices with a cloud server, verifying functionality through an IoT monitoring app.

App Concepts

Student Health and Wellness App Concept

A mobile app concept designed to enhance student health and wellness services, starting with user research and progressing through wireframes and prototypes. The app included features focused on user experience and data privacy, with iterative feedback incorporated into design refinements. A project website was developed to track the app’s progress and document technical decisions.

Animeet

A concept mobile app designed to serve as both a comprehensive database for anime media and a social platform. Worked with a team to research user needs and preferences, and incorporated metadata schemas to structure app functionality. Designed screen mockups in Canva, creating an intuitive interface for users to search and interact with anime content. Presented the app’s concept and design through a polished group presentation.
